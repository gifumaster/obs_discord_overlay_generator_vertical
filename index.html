使い方

最短手順

  1. 共有設定で表示サイズ、間隔、余白、発話演出を決めます。
  2. ユーザー設定でラベル、Discord User ID、表示オフセットを入力します。
  3. 画像ファイル読込または画像貼り付けでアバターを入れ、必要ならトリミングします。
  4. 発話プレビューを切り替えて、見え方を確認します。
  5. 生成CSSをコピーして OBS の Discord Streamkit Overlay に貼り付けます。

JSON保存/読込とローカル下書き再開に対応しています。作業途中の設定も維持できます。
